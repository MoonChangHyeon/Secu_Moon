# Global Components

애플리케이션 전역에서 사용되는 공통 설정, 유틸리티, 예외 처리 클래스 모음입니다.

---

## 📚 구성 (Components)

- **[Config (config)](config/README.md)**: Spring Security, MVC, Batch 등 프레임워크 설정.
- **[Util (util)](util/README.md)**: 날짜 변환, 파일 처리 등 공통 유틸리티.
- **[Exception (exception)](exception/README.md)**: Global Exception Handler 및 커스텀 예외.
