## 📚 모듈 목록 (Module List)

- **[Fortify (fortify)](fortify/README.md)**: Fortify 취약점 점검 및 탐색, 버전 비교.
- **[Compliance (compliance)](compliance/README.md)**: 보안 규정(ISMS-P, ISO27001 등) 매핑 및 뷰어.
- **[Analysis (analysis)](analysis/README.md)**: 소스코드 취약점 분석 요청 및 실행, 결과 조회.
- **[SBOM (sbom)](sbom/README.md)**: 오픈소스 라이선스 및 컴포넌트 분석.
- **[User (user)](user/README.md)**: 사용자 계정 및 권한 관리.
- **[Settings (settings)](settings/README.md)**: 시스템 환경 설정.
- **[Mocha (mocha)](mocha/README.md)**: 레거시/외부 DB 연동.
