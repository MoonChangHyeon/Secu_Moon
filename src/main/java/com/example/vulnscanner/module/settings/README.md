# Settings Module

시스템 환경 설정 및 전역 변수를 관리하는 모듈입니다.

---

## 🔥 주요 기능 (Features)

### 1. 분석 도구 설정
- Fortify SSC/SCA 경로 및 파라미터 설정.
- 분석 타임아웃 등 실행 옵션 관리.

### 2. 시스템 환경 변수
- 파일 저장 경로, 로그 레벨 등 애플리케이션 동작에 필요한 설정값 관리.

---

## 🛠 기술 명세 (Technical Specs)

- **Controller**: `SettingsController`.
- **Entity**: `SystemSetting`.
