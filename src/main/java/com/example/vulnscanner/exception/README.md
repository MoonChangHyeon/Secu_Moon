# 예외 처리 (Exception)

이 디렉토리는 애플리케이션 전역에서 발생하는 예외를 공통으로 처리하기 위한 클래스들을 포함합니다.

## 파일 목록 및 설명

### `GlobalExceptionHandler.java`
- **역할**: 전역 예외 처리기 (`@ControllerAdvice`).
- **주요 기능**:
  - 컨트롤러 전반에서 발생하는 특정 예외를 가로채서 통일된 응답을 반환합니다.
  - **`MaxUploadSizeExceededException` 처리**: 파일 업로드 용량 초과 시, 사용자에게 친절한 JSON 형태의 에러 메시지("파일 크기가 너무 큽니다 ...")를 반환하여 프론트엔드에서 알림창(Alert)을 띄울 수 있도록 지원합니다.
